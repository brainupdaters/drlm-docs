DRLM TFTPD
==========

TFTP daemon server for DRLM

Management
~~~~~~~~~~

Start DRLM TFTPD

.. code-block:: console
 
  ~# systemctl start drlm-tftpd.service

Restart DRLM TFTPD

.. code-block:: console

  ~# systemctl restart drlm-tftpd.service

Stop DRLM TFTPD

.. code-block:: console

  ~# systemctl stop drlm-tftpd.service
